# Modelo de negocio

## Cómo ganamos dinero

Hay dos formas de usar GuideFlow. El guía elige la que le va mejor:

### Plan Gratuito (para empezar)

- **Cuota mensual:** 0€
- **Lo que nos llevamos:** Un 5% de cada reserva que se pague
- **Ejemplo:** Si un cliente paga 60€, nos quedamos 3€ y el guía recibe 56.10€ (Stripe cobra ~0.90€ aparte)

### Plan Pro (para guías con volumen)

- **Cuota mensual:** 19€
- **Lo que nos llevamos:** 0€ de comisión por reserva
- **Ejemplo:** Si un cliente paga 60€, el guía recibe 59.10€ (solo paga el fee del banco)

## ¿Cuándo le sale rentable el Plan Pro al guía?

| Reservas al mes | Paga con Plan Gratuito (5%) | Paga con Plan Pro (19€/mes) | Mejor opción |
|----------------|---------------------------|---------------------------|-------------|
| 200€ | 10€ | 19€ | Gratuito |
| 380€ | 19€ | 19€ | Da igual |
| 600€ | 30€ | 19€ | **Pro** |
| 1.000€ | 50€ | 19€ | **Pro (obvio)** |

**Punto de cruce: 380€/mes en reservas.** A partir de ahí, la comisión supera la cuota.

Le mostraremos al guía un aviso automático: *"Este mes has pagado 25€ en comisiones. Con el Plan Pro pagarías 19€. ¿Quieres cambiar?"*

## ¿Qué pasa si un guía tiene 0 reservas un mes?

- **Plan Gratuito:** No paga nada. No perdemos nada.
- **Plan Pro:** Sigue pagando 19€ (pero puede cancelar cuando quiera).

## Plan anual

19€/mes → **15€/mes** si paga anualmente (180€/año). Ahorra 48€ y nosotros mejoramos el flujo de caja.

## ¿Qué nos dice si el modelo funciona?

| Señal | Significado |
|-------|-----------|
| Guías convierten de gratis a Pro | El producto les aporta valor real |
| Churn < 10% mensual | No se van → estamos resolviendo su problema |
| MRR crece mes a mes | El negocio escala |
| Reservas por guía suben | Usan más la plataforma con el tiempo |
