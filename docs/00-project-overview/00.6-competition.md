# La competencia

## ¿Qué usan los guías hoy?

La realidad es que la mayoría de los guías **no usan ningún software de reservas**. Su sistema es:

> **Instagram DMs + WhatsApp + libreta + bizum**

Eso es nuestra mayor competencia. No Reservio. No TuriTop. Es la costumbre.

## Alternativas que existen

| Herramienta | Qué es | Precio | ¿Sirve para guías? |
|------------|--------|--------|-------------------|
| **Reservio** | Reservas genéricas (peluquerías, clínicas) | 0-75€/mes | No entiende grupos ni plazas |
| **TuriTop** | Reservas de actividades turísticas | Desde 49€/mes | Pensado para empresas, caro para un autónomo |
| **Anolla** | Reservas outdoor para empresas | A consultar | Demasiado complejo, pensado para flotas de guías |
| **Koalendar** | Agenda de citas online | 0-15€/mes | No tiene pagos ni gestión de plazas |
| **Calendly** | Agenda de reuniones | 0-16€/mes | No tiene pagos ni se adapta a actividades |
| **Google Calendar + Bizum** | El "sistema" de muchos guías | Gratis | Funciona, pero es manual y lleno de errores |

## Nuestra ventaja en UNA frase

> **Calendly + pagos + gestión de plazas, específico para actividades outdoor, por 0€ o 19€/mes.**

## ¿Y si un competidor nos copia?

Nuestra defensa es la **velocidad de ejecución** + la **comunidad de guías**. Quien tenga 100 guías activos primero, gana. Los efectos de red hacen que cada guía que comparte su link sea publicidad gratuita para nosotros.
