# Modelo de Ingresos — GuideFlow

## Modelo Híbrido

### Plan Starter (Gratuito)

- **Precio:** 0€/mes
- **Comisión:** 5% por reserva completada
- **Ideal para:** Captación masiva, validación del producto

### Plan Pro

- **Precio:** 19€/mes
- **Comisión:** 0% (Stripe cobra su fee estándar ~1.4% + 0.25€)
- **Ideal para:** Guías con volumen que buscan predictibilidad

## Lógica de conversión Starter → Pro

El punto de inflexión es **~380€/mes en reservas**:

- 5% de 380€ = 19€ → mismo coste que el Plan Pro
- A partir de ahí, Pro es más barato
- Sistema muestra automáticamente un banner de conversión

## Flujo de dinero (Stripe Connect)

```
Cliente paga 60€
  └─► Stripe cobra fee (~0.90€)
      └─► Plan Starter: GuideFlow cobra 5% (3€), guía recibe 56.10€
      └─► Plan Pro: GuideFlow cobra 0€, guía recibe 59.10€
          (+ 19€/mes de suscripción aparte)
```

## Estacionalidad

| Temporada | Impacto | Mitigación |
|-----------|---------|------------|
| Verano (jun-sep) | Alta demanda | Guías facturan más → más conversión a Pro |
| Invierno (dic-feb) | Baja demanda | Guías con raquetas/esquí mantienen actividad |
| Temporada baja | Guías pueden cancelar Pro | Ofrecer plan anual con descuento para retener |
