# API Endpoints — GuideFlow

## Auth

| Método | Ruta | Auth | Descripción |
|--------|------|------|-------------|
| POST | `/api/auth/register` | ❌ | Registro de guía |
| POST | `/api/auth/login` | ❌ | Login con email |
| GET | `/api/auth/callback/google` | ❌ | OAuth Google |
| POST | `/api/auth/logout` | ✅ | Cerrar sesión |

## Guía (protegido)

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/guide/profile` | Datos del guía logueado |
| PUT | `/api/guide/profile` | Actualizar perfil |
| POST | `/api/guide/stripe/connect` | Iniciar onboarding Stripe |
| GET | `/api/guide/stripe/status` | Estado cuenta Stripe |
| GET | `/api/guide/dashboard` | Resumen (reservas, ingresos) |

## Actividades (protegido)

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/activities` | Listar actividades del guía |
| POST | `/api/activities` | Crear actividad |
| PUT | `/api/activities/:id` | Editar actividad |
| DELETE | `/api/activities/:id` | Desactivar actividad |

## Slots (protegido)

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/activities/:id/slots` | Listar slots |
| POST | `/api/activities/:id/slots` | Crear slot(s) |
| DELETE | `/api/slots/:id` | Eliminar slot |

## Bookings (protegido)

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/bookings` | Listar reservas (paginado) |
| PUT | `/api/bookings/:id/cancel` | Cancelar y reembolsar |

## Público (sin auth)

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/public/:slug` | Datos del microsite |
| GET | `/api/public/:slug/activities` | Actividades publicadas |
| GET | `/api/public/:slug/slots?date=` | Slots disponibles |
| POST | `/api/public/checkout` | Crear sesión Stripe |

## Webhooks

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/webhooks/stripe` | Webhooks Stripe (verificados) |

## Paginación

Todos los listados usan cursor-based pagination:

```
GET /api/bookings?cursor=abc123&limit=20
```
