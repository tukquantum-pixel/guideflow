## Justificaciones clave

### ¿Por qué Docker + PostgreSQL en vez de Supabase?

- **Control total:** No dependemos de un servicio tercero
- **Experiencia:** Ya dominamos Docker + PostgreSQL en producción
- **Coste:** 5€/mes VPS vs tiers gratuitos con límites
- **Portabilidad:** Nos movemos a otro proveedor cuando queramos

### ¿Por qué Auth.js en vez de un BaaS?

- **Sin dependencias** de servicios externos para auth
- **JWT + sesiones** con PostgreSQL directamente
- **Google OAuth** integrado out-of-the-box
