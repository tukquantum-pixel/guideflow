# Documentación de Mapas — GuideFlow

## Estado actual (MVP)

- ✅ `MapPicker` para creación de actividades (clic, arrastrar, geocodificar)
- ✅ `MapView` para visualización pública (solo lectura)
- ✅ Geocodificación dirección → coordenadas (Nominatim)
- ✅ Pin arrastrable con actualización `onDragEnd` (no en cada pixel)
- ✅ Campos `meetingLat` / `meetingLng` en Activity (schema Prisma)
- ✅ Sin costes, sin API keys

## Stack técnico

| Componente | Tecnología | Coste |
|---|---|---|
| Mapa | Leaflet + OpenStreetMap tiles | 0€ |
| Geocodificación | Nominatim (OSM) | 0€ |
| Pin | SVG custom → divIcon musgo (#5B8C5A) | — |
| SSR | `next/dynamic` con `ssr: false` | — |
| CSS | CDN unpkg (leaflet@1.9.4) | — |

## Limitaciones conocidas

| Limitación | Mitigación implementada |
|---|---|
| Nominatim: 1 req/sec | Throttle de 1.1s entre peticiones |
| Geocodificación puede fallar | Mensaje claro + opción de pin manual |
| Sin autocompletar | Botón de búsqueda explícito |
| Sin rutas GPX | Solo punto de encuentro (fase 1) |
| Sin capas de desnivel | No MVP |
