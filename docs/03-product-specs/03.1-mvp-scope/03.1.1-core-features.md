# Core Features (En el lanzamiento) — GuideFlow

## Si solo pudiéramos lanzar con 3 funcionalidades

1. **Microsite público del guía** con actividades y calendario
2. **Pago online** con Stripe
3. **Confirmación automática** por email

Todo lo demás es mejora. Pero con estas 3, un guía ya puede cobrar.

## Lista completa del lanzamiento

### 1. Registro y onboarding del guía

- Registro con email o Google
- Wizard de 3 pasos: datos → Stripe → primera actividad
- Slug personalizado (`@nombre-guia`)

### 2. Gestión de actividades

- Crear/editar/borrar actividades (nombre, descripción, precio, duración)
- Fotos (hasta 5)
- Plazas máximas por slot
- Marcar como "dependiente de meteo"

### 3. Motor de disponibilidad

- Calendario con slots por día/hora
- Bloqueo automático al llenarse
- Reglas recurrentes (ej: todos los sábados a las 9:00)

### 4. Microsite público

- Página pública responsive y profesional
- SEO optimizado
- QR code descargable

### 5. Checkout y pagos

- Pago online vía Stripe
- Split automático (5% en plan Starter)
- Confirmación instantánea

### 6. Dashboard del guía

- Calendario de próximas actividades
- Lista de reservas con datos del cliente
- Ingresos del mes

### 7. Comunicación automática

- Email de confirmación al reservar
- Recordatorio 24h antes
- Notificación al guía de nueva reserva

### 8. Cancelaciones

- Política configurable por el guía
- Reembolso automático según política
