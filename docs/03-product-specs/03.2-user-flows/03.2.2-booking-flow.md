# Flujo de Reserva (Cliente Final) — GuideFlow

## Clicks para pagar: 4

```
1. Entra al link (desde Instagram/WhatsApp)
2. Elige actividad
3. Elige fecha/hora + rellena datos
4. Paga con tarjeta
→ Confirmación instantánea
```

## Sin registro. Sin app. Sin contraseñas

| Campo | Obligatorio | Motivo |
|-------|------------|--------|
| Nombre completo | ✅ | Identificación |
| Email | ✅ | Confirmación + recordatorio |
| Teléfono | ✅ | Emergencia en montaña |
| Nº personas | ✅ | Control de plazas |
| Comentarios | ❌ | Alergias, nivel, notas |

## Si no hay disponibilidad en la fecha que quiere

- Mostramos las próximas 3 fechas disponibles
- Opción: "Avísame si se abre hueco en esta fecha"

## Si se arrepiente después de pagar

Se aplica la política de cancelación del guía:

- **Flexible:** reembolso total hasta 24h antes
- **Moderada:** reembolso del 50% hasta 48h antes
- **Estricta:** sin reembolso
