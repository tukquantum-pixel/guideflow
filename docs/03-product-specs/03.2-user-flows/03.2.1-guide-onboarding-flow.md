# Flujo de Onboarding del Guía — GuideFlow

## Pasos mínimos para tener el link funcionando: 3

```
Paso 1: Datos Básicos (2 min)
  ├─ Nombre / nombre artístico
  ├─ Email (ya rellenado si usa Google)
  ├─ Teléfono
  ├─ Slug personalizado (@nombre)
  └─ Zona geográfica

Paso 2: Conectar Stripe (3 min)
  ├─ Botón "Conectar cuenta de cobro"
  ├─ Redirect a Stripe Connect Express
  └─ Redirect de vuelta automáticamente

Paso 3: Primera Actividad (5 min)
  ├─ Nombre de la actividad
  ├─ Precio por persona
  ├─ Duración
  ├─ Plazas máximas
  └─ Primer slot de disponibilidad

→ ¡LISTO! Tu link es: app.guideflow.com/@tu-nombre
```

## Feedback durante el proceso

- **Barra de progreso** visible: "Paso 2 de 3"
- **Preview en vivo** de cómo quedará su microsite
- **Mensaje de éxito** al terminar con su link para copiar y compartir

## Puntos de abandono y cómo los evitamos

| Punto de abandono | Frecuencia esperada | Solución |
|-------------------|--------------------| ---------|
| Registro → Paso 1 | Baja | Pedir solo lo mínimo |
| Paso 2 → Stripe KYC | **Alta** | Guía paso a paso, "solo necesitas tu DNI y una cuenta bancaria" |
| Paso 3 → Primera actividad | Media | Plantillas predefinidas ("Senderismo", "Barranquismo", etc.) |
