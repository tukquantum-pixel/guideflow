## 📊 Comparación de protocolos

| Feature | HTTP/1.1 | HTTP/2 | HTTP/3 (QUIC) |
|---------|----------|--------|---------------|
| Multiplexing | ❌ | ✅ | ✅ |
| Header compression | ❌ | HPACK | QPACK |
| Server push | ❌ | ✅ | ❌ (deprecated) |
| Connection setup | 1-3 RTT | 1-2 RTT | 0-1 RTT |
| Head-of-line blocking | Sí | Parcial | No |
| Protocolo base | TCP | TCP | UDP (QUIC) |

## ⚙️ Activar HTTP/2 (Nginx)

```nginx
server {
  listen 443 ssl http2;
  ssl_protocols TLSv1.2 TLSv1.3;
}
```

## ⚙️ Activar HTTP/3 (Cloudflare)

- En Cloudflare Dashboard → Speed → Optimization → HTTP/3 (QUIC) → ON
- Automático si el navegador lo soporta (Chrome, Firefox, Safari)

## 🎯 Beneficio para GuideFlow

- Múltiples recursos paralelos (tiles de mapa + API + assets)
- Menor latencia en conexiones móviles (QUIC = 0-RTT)
