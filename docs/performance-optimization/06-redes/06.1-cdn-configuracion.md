## ⚙️ Cloudflare Setup

1. Añadir dominio → cambiar nameservers
2. Activar proxy (nubecita naranja) para subdominio `app.guideflow.com`
3. SSL → Full (strict)
4. Page Rules:
   - `/_next/static/*` → Cache Level: Standard, Edge TTL: 1 month
   - `/api/*` → Cache Level: Bypass
   - `/*.webp` → Cache Level: Standard, Edge TTL: 30 days

## 📊 Headers de CDN

```
# Assets estáticos (inmutables por hash)
Cache-Control: public, max-age=31536000, immutable

# HTML páginas ISR
Cache-Control: public, s-maxage=3600, stale-while-revalidate=86400

# API responses públicas
Cache-Control: public, s-maxage=300, stale-while-revalidate=3600

# Datos privados (NUNCA cachear en CDN)
Cache-Control: private, no-store
```
