## 📊 Brotli vs Gzip

| Aspecto | Gzip | Brotli |
|---------|------|--------|
| Ratio compresión | ~70% | ~77% |
| Velocidad compresión | Rápida | Media-lenta |
| Velocidad descompresión | Rápida | Rápida |
| Soporte navegadores | 99%+ | 96%+ |
| Soporte Cloudflare | ✅ | ✅ (automático) |

## ⚙️ Cloudflare (automático)

Cloudflare comprime automáticamente con Brotli si el navegador lo soporta (header `Accept-Encoding: br`).

## ⚙️ Self-hosted (Nginx)

```nginx
# Requiere módulo ngx_brotli
brotli on;
brotli_comp_level 6;  # 1-11, 6 = buen balance
brotli_types text/plain text/css application/json application/javascript text/xml application/xml;
```

| Recurso | Sin compresión | Gzip | Brotli | Ahorro vs gzip |

| API JSON 30kb | 30kb | 8kb | 6kb | 25% menos |
