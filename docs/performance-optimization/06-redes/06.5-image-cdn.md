## 🎯 Recomendación: Cloudflare Images

- Cost: ~5$/mes para miles de imágenes
- Transformaciones via URL: `./image/w=400,h=300,fit=cover/image.jpg`
- Auto-format: sirve WebP a Chrome, AVIF a navegadores compatibles
- CDN global incluido

## ⚙️ Integración con Next.js

```typescript
// next.config.ts
images: {
  loader: "custom",
  loaderFile: "./lib/cloudflare-image-loader.ts",
}

// lib/cloudflare-image-loader.ts
export default function cfLoader({ src, width, quality }: any) {
  return `https://imagedelivery.net/<account>/${src}/w=${width},q=${quality || 80}`
}
```
