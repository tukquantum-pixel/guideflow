## ⚙️ Configuración DNS optimizada

| Registro | Tipo | Valor | TTL |
|----------|------|-------|-----|
| `guideflow.com` | A | IP del servidor | 300s (proxied por CF) |
| `www` | CNAME | `guideflow.com` | 300s |
| `api` | CNAME | `guideflow.com` | 300s |

## 📊 Preconexión DNS en frontend

```html
<!-- En layout.tsx: preconectar a dominios conocidos -->
<link rel="dns-prefetch" href="https://fonts.googleapis.com" />
<link rel="dns-prefetch" href="https://js.stripe.com" />
<link rel="preconnect" href="https://js.stripe.com" crossOrigin="anonymous" />
```

## 🎯 Impacto

- DNS lookup: 20-120ms (sin prefetch) → 0ms (con prefetch)
- Cloudflare DNS: ~11ms (uno de los más rápidos del mundo)
