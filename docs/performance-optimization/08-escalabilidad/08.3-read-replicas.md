# 08.3 – Read Replicas

| Metadata | Valor |
|----------|-------|

## 🎯 Qué es

Copia de la BD principal (primary) que recibe todas las escrituras. Las réplicas solo leen. Distribuye carga de lectura.

## 📊 Cuándo necesario

- Ratio lectura/escritura: 90/10 (típico en GuideFlow → buen candidato)
- Primary saturado en lecturas
- A partir de ~10k usuarios activos

## ⚙️ Implementación con Prisma

```typescript
// prisma/schema.prisma (Prisma 6)
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")  // Primary (read-write)
  directUrl = env("DIRECT_URL")   // Sin proxy
}

// Para réplica de lectura, usar Prisma Client Extension
const prisma = new PrismaClient().$extends({
  query: {
    $allOperations({ operation, args, query }) {
      // Dirigir lecturas a réplica
      if (["findMany", "findUnique", "count", "aggregate"].includes(operation)) {
        return queryOnReplica(args)
      }
      return query(args) // Escrituras al primary
    }
  }
})
```
