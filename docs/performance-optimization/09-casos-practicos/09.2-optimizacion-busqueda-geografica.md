## 🎯 Problema

Buscar rutas en un radio de X km desde una coordenada. Sin índice espacial, escanea todas las filas.

## ⚙️ Solución: PostGIS + GIST index

```sql
-- 1. Activar extensión
CREATE EXTENSION IF NOT EXISTS postgis;

-- 2. Índice espacial
CREATE INDEX idx_tracks_geo ON "Track"
  USING GIST (ST_SetSRID(ST_Point("startLng", "startLat"), 4326)::geography);

-- 3. Query optimizada: rutas en 20km
SELECT id, name, distance FROM "Track"
WHERE ST_DWithin(
  ST_SetSRID(ST_Point("startLng", "startLat"), 4326)::geography,
  ST_SetSRID(ST_Point(:userLng, :userLat), 4326)::geography,
  20000  -- 20km en metros
)
ORDER BY distance DESC LIMIT 20;
```

## 📊 Impacto

| Métrica | Sin PostGIS | Con PostGIS |
|---------|-------------|-------------|
| Query time (500 rutas) | 300ms | 5ms |
| Query time (10k rutas) | 3s | 8ms |
| Método | Haversine en app | GIST index en BD |
