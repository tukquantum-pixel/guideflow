## 🎯 Cuándo necesario

A partir de ~5k-10k usuarios activos diarios o cuando una sola instancia no pueda manejar el pico de tráfico.

## ⚙️ Opciones

| Solución | Coste | Complejidad | Cuándo |
|----------|-------|-------------|--------|
| Nginx upstream | 0€ | Baja | Self-hosted, 2-3 instancias |
| Cloudflare Load Balancer | 5$/mes | Baja | Si ya usamos CF |
| AWS ALB / DO Load Balancer | 15-20$/mes | Media | Cloud managed |

## 📊 Nginx upstream config

```nginx
upstream guideflow {
  least_conn;  # Enviar al servidor con menos conexiones
  server app1:3000 weight=1;
  server app2:3000 weight=1;
  server app3:3000 weight=1 backup;
}

server {
  location / { proxy_pass http://guideflow; }
}
```

## 🎯 Sticky sessions (si necesario)

```nginx
upstream guideflow {
  ip_hash;  # Misma IP → mismo servidor (para WebSockets)
  server app1:3000;
  server app2:3000;
}
```
