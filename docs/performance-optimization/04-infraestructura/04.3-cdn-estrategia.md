## ⚙️ Qué servir desde CDN

| Recurso | CDN | Caché | Justificación |
|---------|-----|-------|---------------|
| JS/CSS bundles | ✅ | 1 año (immutable) | Hashed filenames |
| Imágenes estáticas | ✅ | 30 días | Cambian poco |
| Fotos de usuarios | ✅ | 7 días | Cloudflare Images |
| API responses | ⚠️ | Solo públicas, 5min | Con headers Cache-Control |
| HTML páginas | ⚠️ | 5min (stale-while-revalidate) | Para ISR pages |

## 🎯 Reglas de caché en Cloudflare

```
Page Rule: /_next/static/* → Cache Level: Standard, Edge TTL: 1 month
Page Rule: /api/public/* → Cache Level: Standard, Edge TTL: 5 min
Page Rule: /api/user/* → Cache Level: Bypass (nunca cachear datos privados)
```
