## ⚙️ Alertas críticas

| Alerta | Condición | Canal |
|--------|-----------|-------|
| Servidor caído | Healthcheck KO 3x seguidas | Telegram + email |
| CPU >90% | 5 min sostenido | Slack |
| Disco >85% | Threshold | Email |
| Certificado SSL expira | <14 días | Email |
| BD conexiones >80% | Threshold | Slack |

## 🎯 Dashboard mínimo viable

- CPU / RAM / Disco (host)
- Requests/s y latencia p50/p95 (app)
- Conexiones BD activas (PostgreSQL)
- Error rate 5xx (app)
