## 🎯 Cuándo implementar

Fase 3 (+50k usuarios). Antes de eso, escalar manualmente es más eficiente y barato.

## 📊 Triggers de scaling

| Métrica | Scale up | Scale down |
|---------|----------|------------|
| CPU | >70% durante 5min | <30% durante 15min |
| Memoria | >80% durante 5min | <40% durante 15min |
| Requests/s | >500 rps por instancia | <100 rps |
| Response time p95 | >1s durante 3min | <200ms durante 15min |

## ⚙️ Opciones por plataforma

| Plataforma | Mecanismo | Min-Max instancias |
|-----------|-----------|-------------------|
| Vercel | Automático (serverless) | 0-∞ |
| Docker Swarm | `replicas` + health checks | 1-5 |
| Kubernetes (HPA) | HorizontalPodAutoscaler | 2-10 |
| AWS ECS | Target tracking scaling | 1-10 |

## 🎯 Recomendación para GuideFlow

- **Año 1**: instancia única, scale up manual
- **Año 2**: 2 instancias + Nginx LB
- **Año 3+**: Kubernetes o Vercel Pro
