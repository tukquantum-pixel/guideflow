## 🎯 Qué es

Seguimiento end-to-end de una request a través de todos los servicios. Para GuideFlow (monolito), trazar request → caché → BD → response.

## ⚙️ Implementación con Sentry Performance

```typescript
import * as Sentry from "@sentry/nextjs"

// Trace automático en API routes
export async function GET(req: Request) {
  return Sentry.withScope(scope => {
    scope.setTag("endpoint", "/api/routes")
    // La request se traza automáticamente
    return Response.json(data)
  })
}
```

## 📊 Cuándo usar

| Escenario | Herramienta | Coste |
|-----------|-------------|-------|
| App Next.js | Sentry Performance | Gratis (5k events/mes) |
| Microservicios (futuro) | Jaeger + OpenTelemetry | Self-hosted gratis |
| En Vercel | Vercel Observability | Incluido en Pro |
