## 📊 Dashboards necesarios

### 1. Overview (vista general)

| Panel | Tipo | Datos |
|-------|------|-------|
| Requests/s | Gráfico línea | Prometheus `http_requests_total` |
| Latencia p50/p95/p99 | Gráfico línea | `http_request_duration_seconds` |
| Error rate | Gauge | % de 5xx sobre total |
| Uptime | Stat | Uptime Robot API |

### 2. Base de datos

| Panel | Tipo | Datos |
|-------|------|-------|
| Conexiones activas | Gauge | `pg_stat_activity` |
| Queries/s | Gráfico | `pg_stat_statements` |
| Cache hit ratio | Gauge | `pg_stat_database` |
| Tabla más grande | Tabla | `pg_total_relation_size` |

### 3. Negocio

| Panel | Tipo | Datos |
|-------|------|-------|
| Usuarios activos/hoy | Stat | Custom query |
| Suscripciones activas | Stat | Prisma count |
| Rutas vistas/hora | Gráfico | App metrics |
| Revenue MTD | Stat | Stripe API |
