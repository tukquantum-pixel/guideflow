## ⚙️ Alerta de coste

```typescript
// Cron job diario (03:00 UTC)
async function checkDailyCost() {
  const cost = await getDailyCostEstimate()
  const average = await get30DayAverage()
  if (cost > average * 1.5) {
    await sendAlert(`⚠️ Coste diario alto: ${cost}€ (media: ${average}€)`)
  }
}
```
