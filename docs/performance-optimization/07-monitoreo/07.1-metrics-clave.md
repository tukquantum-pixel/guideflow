## ⚙️ Implementación con Prometheus

```typescript
import promClient from "prom-client"

const httpDuration = new promClient.Histogram({
  name: "http_request_duration_seconds",
  help: "Duración de peticiones HTTP",
  labelNames: ["method", "route", "status"],
  buckets: [0.05, 0.1, 0.3, 0.5, 1, 2, 5],
})

// Endpoint /api/metrics para Prometheus
export async function GET() {
  return new Response(await promClient.register.metrics(), {
    headers: { "Content-Type": promClient.register.contentType },
  })
}
```
