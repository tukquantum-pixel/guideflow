## 📊 Clasificación de páginas GuideFlow

| Página | Tipo | Justificación |
|--------|------|---------------|
| Landing `/` | SSG + ISR (1h) | SEO critical, cambia poco |
| `/explorar` | SSR + cache | Búsqueda dinámica con filtros |
| `/ruta/[id]` | ISR (1h) | SEO + datos semi-estáticos |
| `/premium` | Client-side | Interactividad Stripe, no necesita SEO server |
| `/dashboard` | SSR | Datos personalizados, sin caché pública |
| `/[slug]` (guía) | ISR (30min) | SEO critical, datos del guía |
| `/cuenta` | SSR | Datos privados del usuario |

## ⚙️ Patrones de configuración

```typescript
// ISR: regenerar cada hora
export const revalidate = 3600

// SSR puro: sin caché
export const dynamic = "force-dynamic"

// SSG: generar en build time
export async function generateStaticParams() {
  const guides = await prisma.guide.findMany({ select: { slug: true } })
  return guides.map(g => ({ slug: g.slug }))
}
```

## 🎯 Regla general

- **Tiene SEO** → SSG o ISR
- **Tiene datos personalizados** → SSR
- **Solo interactividad** → Client-side
