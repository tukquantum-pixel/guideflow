## ⚙️ Setup

```bash
npm install @next/bundle-analyzer
```

```typescript
// next.config.ts
const withBundleAnalyzer = require("@next/bundle-analyzer")({ enabled: process.env.ANALYZE === "true" })
module.exports = withBundleAnalyzer({ /* config */ })
```

## 🎯 Reglas

1. **Librerías >50kb** → siempre lazy import
2. **Tree-shaking** → importar funciones específicas, no `*`
3. **Dependencias duplicadas** → auditar con `npm ls <package>`
4. **Análisis mensual** → ejecutar `ANALYZE=true npm run build` cada sprint
