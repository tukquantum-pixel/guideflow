## 📊 Métricas objetivo

| Métrica | Objetivo | Cómo mejorarla |
|---------|----------|----------------|
| **LCP** (Largest Contentful Paint) | < 2.5s | Optimizar imágenes, CDN, eliminar recursos bloqueantes |
| **INP** (Interaction to Next Paint) | < 200ms | Reducir JS, lazy loading, evitar tareas largas |
| **CLS** (Cumulative Layout Shift) | < 0.1 | Dimensiones fijas para imágenes, evitar inyección dinámica |
