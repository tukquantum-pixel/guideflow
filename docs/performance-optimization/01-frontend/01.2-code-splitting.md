## 🎯 Estrategia por componentes pesados

```tsx
// Componentes pesados → dynamic import
import dynamic from "next/dynamic"

const MapView = dynamic(() => import("@/components/map-view"), {
  ssr: false,
  loading: () => <div className="h-64 bg-roca animate-pulse rounded" />
})

// Uso: solo se carga cuando se renderiza
{showMap && <MapView center={coords} />}
```

## 📦 Estrategia por librerías

```tsx
// ❌ MAL: importar toda la librería
import * as turf from "@turf/turf" // 200kb

// ✅ BIEN: importar solo lo necesario
import distance from "@turf/distance" // 20kb

// ✅ MEJOR: lazy import cuando se necesita
const calcDistance = async (a: any, b: any) => {
  const { default: distance } = await import("@turf/distance")
  return distance(a, b)
}
```

## 🎯 Regla: páginas que NO necesitan SSR

| Página | SSR | Motivo |
|--------|-----|--------|
| `/premium` | No | Contenido estático, client interactivity |
| `/explorar` (mapa) | No | Leaflet no funciona en server |
| `/dashboard` | Sí | Datos personalizados |
| Landing `/` | Sí | SEO critical |
