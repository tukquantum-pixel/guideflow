## 📦 Next.js Image Component (obligatorio)

```tsx
import Image from "next/image"

<Image
  src="/ruta-foto.jpg" alt="Descripción"
  width={800} height={600}
  sizes="(max-width: 768px) 100vw, (max-width: 1200px) 50vw, 33vw"
  priority={false} loading="lazy"
/>
```

## ⚙️ Configuración en next.config.ts

```typescript
images: {
  formats: ["image/avif", "image/webp"],
  deviceSizes: [640, 750, 828, 1080, 1200, 1920],
  imageSizes: [16, 32, 48, 64, 96, 128, 256, 384],
}
```

## 📊 Estrategia por tipo de imagen

| Tipo | Estrategia | priority |
|------|-----------|----------|
| Hero de página | `priority={true}`, fetchPriority="high" | Alta |
| Fotos de rutas | `loading="lazy"`, sizes correctos | Media |
| Avatares | `loading="lazy"`, 64x64px | Baja |
| Galerías | `loading="lazy"`, Intersection Observer | Baja |

## 🎯 Fotos subidas por usuarios

- Subir a CDN (Cloudflare Images / Cloudinary)
- Generar variantes: thumbnail (200px), medium (800px), large (1600px)
- Formato WebP/AVIF automático
- Compresión server-side: max 2000px, calidad 80%
