## ⚙️ Performance budget en CI

```yaml
# .github/workflows/lighthouse.yml
- name: Lighthouse CI
  run: |
    npx lhci autorun --config=lighthouserc.json
  # Falla si LCP > 2.5s o CLS > 0.1
```

## 🎯 Alertas

- LCP > 3s en producción → alerta Slack
- Bundle size > 200kb → bloquear PR
- Error rate > 1% → investigar inmediatamente
