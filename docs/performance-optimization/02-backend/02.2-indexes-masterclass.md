## 📊 Índices actuales en schema.prisma

| Modelo | Índice | Columnas |
|--------|--------|----------|
| Guide | `@@index([slug])` | slug |
| Activity | `@@index([guideId, active])` | guideId + active |
| TimeSlot | `@@index([activityId, date, status])` | compuesto |
| Booking | `@@index([guideId, createdAt])` | guideId + createdAt |
| Stage | `@@index([trackId, order])` | trackId + order |
| Checkpoint | `@@index([stageId, order])` | stageId + order |
| Location | `@@index([lat, lng])` | coordenadas |

## ⚙️ Índices recomendados a añadir

```sql
-- Búsqueda geográfica (PostGIS)
CREATE INDEX idx_locations_geo ON "Location" USING GIST (
  ST_SetSRID(ST_Point(lng, lat), 4326)::geography
);

-- Búsqueda de rutas por distancia/desnivel
CREATE INDEX idx_tracks_stats ON "Track" (distance, "elevationGain")
  WHERE distance IS NOT NULL;

-- Búsqueda full-text en actividades
CREATE INDEX idx_activities_search ON "Activity"
  USING GIN (to_tsvector('spanish', title || ' ' || COALESCE(description, '')));
```
