## 📊 Formatos de compresión

| Formato | Ratio | Soporte | Prioridad |
|---------|-------|---------|-----------|
| Brotli (br) | ~30% mejor que gzip | Chrome, Firefox, Safari, Edge | ⭐ Preferido |
| Gzip | Baseline | Universal | Fallback |
| Sin compresión | 0% | - | Solo imágenes ya comprimidas |

## ⚙️ Configuración

En Vercel/Next.js la compresión es automática. Para self-hosted:

```typescript
// next.config.ts
module.exports = { compress: true } // gzip automático

// Para Brotli, configurar en Nginx:
// gzip on; gzip_types text/plain application/json text/css application/javascript;
// brotli on; brotli_types text/plain application/json text/css application/javascript;
```

## 📊 Impacto medido

| Recurso | Sin compresión | Gzip | Brotli |

| JSON API (30kb) | 30kb | 8kb | 6kb |
