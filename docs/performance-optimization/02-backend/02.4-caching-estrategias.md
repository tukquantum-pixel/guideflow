## 📊 Qué cachear y qué no

| Dato | Cachear | TTL | Motivo |
|------|---------|-----|--------|
| Lista de rutas populares | ✅ | 1h | Query pesada, cambia poco |
| Perfil público de guía | ✅ | 30min | Consulta frecuente |
| Resultados de búsqueda | ✅ | 10min | Mismos filtros repetidos |
| Condiciones meteorológicas | ✅ | 1h | API externa costosa |
| Datos de sesión del usuario | ❌ | - | Siempre actualizados |
| Estado de reserva | ❌ | - | Crítico que esté en tiempo real |
| Checkout Stripe | ❌ | - | Seguridad financiera |

## ⚙️ Patrón cache-aside con `unstable_cache`

```typescript
import { unstable_cache } from "next/cache"

export const getGuideProfile = unstable_cache(
  async (slug: string) => {
    return prisma.guide.findUnique({
      where: { slug },
      select: { name: true, bio: true, rating: true, avatarUrl: true }
    })
  },
  ["guide-profile"], // tag para invalidación
  { revalidate: 1800 }
)
```

## 🔄 Invalidación

```typescript
import { revalidateTag } from "next/cache"
// Cuando el guía edita su perfil:
revalidateTag("guide-profile")
```
