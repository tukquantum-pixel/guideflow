## 📊 Métricas clave

| Métrica | Objetivo | Alerta si |
|---------|----------|-----------|
| Hit rate global | >80% | <60% |
| Hit rate Redis | >85% | <70% |
| Hit rate CDN | >90% (estáticos) | <80% |
| Latencia Redis p95 | <5ms | >20ms |
| Memoria Redis | <80% del límite | >90% |
| Evictions/min | <10 | >50 |

## ⚙️ Monitorización Redis

```bash
# Estadísticas en tiempo real
redis-cli INFO stats | grep -E "keyspace_hits|keyspace_misses|evicted_keys"

# Hit rate
redis-cli INFO stats | awk '/hits/ {h=$2} /misses/ {m=$2} END {print h/(h+m)*100 "%"}'
```

## ⚙️ Logging personalizado

```typescript
// Wrapper con métricas
export async function getCachedOrFetch<T>(key: string, fetchFn: () => Promise<T>, ttl = 3600) {
  const start = performance.now()
  const cached = await redis.get<T>(key)
  if (cached) {
    console.log(`[CACHE HIT] ${key} (${(performance.now() - start).toFixed(1)}ms)`)
    return cached
  }
  console.log(`[CACHE MISS] ${key}`)
  const data = await fetchFn()
  await redis.setex(key, ttl, JSON.stringify(data))
  return data
}
```
