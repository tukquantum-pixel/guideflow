# 03.4 – Connection Limits

| Metadata | Valor |
|----------|-------|
| Versión | 1.0 |
| Última actualización | 2026-03-08 |
| Responsable | DBA / Backend |
| Estado | Borrador |

## 📊 Límites por etapa

| Escala | max_connections | Pool (por instancia) | Instancias | Total |
|--------|-----------------|----------------------|------------|-------|
| 1k usuarios | 100 | 10 | 1 | 10 |
| 10k usuarios | 200 | 15 | 2 | 30 |
| 100k usuarios | 300 + PgBouncer | 20 | 5 | 100 |

## ⚙️ PgBouncer (connection pooler externo)

```ini
# pgbouncer.ini
[pgbouncer]
pool_mode = transaction   # Libera conexión entre transacciones
max_client_conn = 1000
default_pool_size = 20
min_pool_size = 5
reserve_pool_size = 5
reserve_pool_timeout = 3
```

## 📏 Monitorizar

```sql
-- Conexiones activas
SELECT state, count(*) FROM pg_stat_activity GROUP BY state;

-- Alerta si >80% del máximo
SELECT count(*) AS active,
  current_setting('max_connections')::int AS max,
  round(count(*)::numeric / current_setting('max_connections')::int * 100) AS pct
FROM pg_stat_activity;
```
