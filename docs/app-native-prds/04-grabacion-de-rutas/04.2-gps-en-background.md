# PRD: GPS en Background

| Metadata | Valor |
|----------|-------|
| **Versión** | 1.0 |
| **Fecha** | 2026-03-10 |
| **Estado** | Borrador |

## 🎯 Objetivo

Permitir la grabación de rutas con la pantalla apagada o la app en segundo plano.

## Librería

`react-native-background-geolocation` (transistorsoft)

## Configuración

```typescript
BackgroundGeolocation.ready({
  desiredAccuracy: BackgroundGeolocation.DESIRED_ACCURACY_HIGH,
  distanceFilter: 10,           // metros mínimo entre puntos
  stopOnTerminate: false,       // seguir si cierran la app
  startOnBoot: false,
  enableHeadless: true,         // Android headless mode
  notification: {
    title: "PATHY grabando ruta",
    text: "Toca para volver a la app"
  },
  locationAuthorizationRequest: 'Always', // iOS
})
```

## Optimización batería

| Situación | Frecuencia GPS | distanceFilter |
|-----------|:-:|:-:|
| Caminando (< 5 km/h) | 5s | 10m |
| Corriendo (5-15 km/h) | 3s | 15m |
| Bici (> 15 km/h) | 2s | 20m |
| Parado (0 km/h, >30s) | 30s | 5m |

## Permisos requeridos

| Plataforma | Permiso | Descripción al usuario |
|-----------|---------|------------------------|
| iOS | Always Location | "Para grabar tu ruta con la pantalla apagada" |
| Android | ACCESS_BACKGROUND_LOCATION | "Para seguir grabando cuando la app esté en segundo plano" |

## 🧪 Tests

| Escenario | Esperado |
|-----------|----------|
| App en background | GPS sigue capturando |
| Pantalla apagada | Notificación visible, grabación continúa |
| Modo avión | Puntos se guardan localmente |
| Batería < 10% | Reducir frecuencia, avisar al usuario |
