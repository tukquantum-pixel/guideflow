## 🎯 Objetivo

Pantalla principal de grabación donde el usuario ve su posición en tiempo real sobre un mapa mientras se dibuja el track recorrido.

## 🧠 Experiencia de Usuario

1. Usuario pulsa tab "Grabar" o "Comenzar ruta" desde detalle
2. Se abre pantalla con mapa centrado en posición actual
3. Punto verde animado marca posición
4. Al moverse, línea verde musgo (#5B8C5A) dibuja el recorrido
5. Stats en vivo abajo: distancia, tiempo, desnivel, velocidad
6. Controles: pausar ⏸️, finalizar ⏹️, centrar 📍

## Componentes

- `RecordingMap` — mapa con polyline del track
- `PositionMarker` — punto verde animado con pulse
- `LiveStats` — 4 stats cards
- `RecordingControls` — pause/stop/center buttons
- `LayerSelector` — street/satellite/topo (compartir sistema unificado)

## 🗃️ Estado Local (Zustand)

```typescript
interface RecordingStore {
  state: 'idle' | 'recording' | 'paused' | 'done'
  points: TrackPoint[]
  startTime: number | null
  elapsed: number
  distance: number
  elevationGain: number
  currentSpeed: number
  start(): void
  pause(): void
  resume(): void
  stop(): void
  addPoint(p: TrackPoint): void
}
```

## ⚙️ Lógica

- Capturar GPS cada 5s o 10m (lo que ocurra primero)
- Filtrar puntos con accuracy > 20m (mala señal)
- Calcular distancia incremental (Haversine)
- Calcular desnivel positivo (solo subidas, ignorar ruido < 3m)
- Guardar en MMKV cada 30s (persistencia crash)
