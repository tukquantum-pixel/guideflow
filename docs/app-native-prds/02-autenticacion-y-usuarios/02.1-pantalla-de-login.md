## 🔌 API

| Endpoint | Método | Uso |
|----------|--------|-----|
| `/api/user-auth/...nextauth` | POST | Login con credentials |

## Token

- Guardar JWT en Keychain (iOS) / EncryptedSharedPreferences (Android)
- Auto-refresh antes de expiración
- Si token inválido → redirigir a login
