## 🎯 Objetivo

Configurar el API client (Axios + React Query) para conectar la app con el backend Next.js existente.

## API Client

```typescript
// src/api/client.ts
const api = axios.create({
  baseURL: Config.API_URL, // https://pathy.com/api
  timeout: 10000,
})
api.interceptors.request.use(addAuthToken)
api.interceptors.response.use(handleSuccess, handleError)
```

## Endpoints reutilizables del backend actual

| Endpoint | Método | App screen |
|----------|--------|-----------|
| `/api/routes/all` | GET | Explorador |
| `/api/tracks` | POST | Subir track |
| `/api/user/recorded-routes` | GET | Mis rutas |
| `/api/user/saved-routes` | GET/POST | Guardar ruta |
| `/api/auth/register` | POST | Registro |
| `/api/profile` | GET/PUT | Perfil |
| `/api/user/subscription` | GET | Premium |
| `/api/marketplace/search` | GET | Búsqueda |

## React Query hooks

```typescript
// src/hooks/useRoutes.ts
export function useRoutes(filters) {
  return useQuery(['routes', filters], () => api.get('/routes/all', { params: filters }))
}
```
