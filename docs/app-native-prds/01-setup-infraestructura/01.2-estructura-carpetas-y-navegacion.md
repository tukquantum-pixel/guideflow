## 🎯 Objetivo

Definir la navegación principal de la app con React Navigation: tabs, stacks, modals.

## 📱 Navegación Principal

```
TabNavigator (bottom tabs)
├── ExplorarStack
│   ├── ExplorarScreen (lista + mapa)
│   └── DetalleRutaScreen
├── GrabarStack
│   ├── GrabarScreen (mapa + controles)
│   └── ResumenRutaScreen
├── PerfilStack
│   ├── PerfilScreen
│   ├── MisRutasScreen
│   └── AjustesScreen
└── LogrosScreen

AuthStack (sin tabs)
├── LoginScreen
├── RegisterScreen
└── ForgotPasswordScreen
```

## Tabs inferiores

| Tab | Icono | Pantalla |
|-----|-------|----------|
| Explorar | CompassIcon | Lista de rutas |
| Grabar | MapIcon (🔴 en grab.) | Grabación GPS |
| Perfil | UserIcon | Perfil + stats |
| Logros | StarIcon | Logros + ranking |
