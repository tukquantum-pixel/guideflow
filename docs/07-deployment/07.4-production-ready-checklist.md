## 🎯 CÓMO USAR ESTE CHECKLIST

1. **Cada responsable** debe verificar SU sección personalmente.
2. **No vale "creo que funciona"**. Hay que probarlo con datos reales.
3. **Cada item debe tener evidencia** (captura, log, o test firmado).
4. Si un item está en rojo, **NO HAY GO-LIVE** hasta que esté verde.
5. La revisión final la hace **alguien que NO HAYA programado esa parte**.
