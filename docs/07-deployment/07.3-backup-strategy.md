# Estrategia de Backups — GuideFlow

## Frecuencia

| Tipo | Frecuencia | Retención |
|------|-----------|-----------|
| pg_dump completo | Diario (3:00 AM) | 30 días |
| pg_dump diferencial | Cada 6 horas | 7 días |

## Almacenamiento

- **Primario:** Object Storage Hetzner (~2€/mes)
- **No en el mismo servidor** (protección ante fallo total del VPS)

## Script de backup

```bash
#!/bin/bash
TIMESTAMP=$(date +%Y%m%d_%H%M%S)
docker exec guideflow-db pg_dump -U guideflow guideflow | gzip > /backups/guideflow_$TIMESTAMP.sql.gz
# Upload a Hetzner Object Storage via rclone
rclone copy /backups/guideflow_$TIMESTAMP.sql.gz hetzner:guideflow-backups/
```

## Restauración

```bash
# 1. Bajar el backup
rclone copy hetzner:guideflow-backups/guideflow_20260315.sql.gz /tmp/
# 2. Restaurar
gunzip /tmp/guideflow_20260315.sql.gz
docker exec -i guideflow-db psql -U guideflow guideflow < /tmp/guideflow_20260315.sql
```

## Tiempo de recuperación estimado: < 30 minutos
