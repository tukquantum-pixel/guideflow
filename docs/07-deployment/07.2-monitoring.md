# Monitorización — GuideFlow

## Alertas configuradas

| Alerta | Herramienta | Acción |
|--------|-----------|--------|
| Servidor caído (HTTP 5xx) | UptimeRobot (gratis) | Email + Telegram |
| Uso de disco > 80% | Script cron | Email |
| Stripe webhook falla | Stripe Dashboard | Email automático |
| Error rate > 5% | Logs Docker | Revisar manualmente |

## Logs

```bash
# Ver logs en tiempo real
docker compose logs -f app

# Ver últimos errores
docker compose logs app | grep ERROR

# Logs de PostgreSQL
docker compose logs db
```

## Healthcheck endpoint

```
GET /api/health → 200 OK { "status": "ok", "db": "connected", "stripe": "live" }
```
