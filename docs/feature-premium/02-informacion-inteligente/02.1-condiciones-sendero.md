# 02.1 – Condiciones del Sendero

## 🎯 Descripción funcional

Muestra condiciones meteorológicas y del terreno en tiempo real para cada ruta: temperatura, probabilidad de lluvia/nieve, velocidad del viento, índice UV, calidad del aire y estado del sendero (seco, embarrado, helado, nevado). Datos actualizados cada hora desde APIs externas.

## 🧠 Problema que resuelve

- "Llegué y había nieve sin esperarlo, no llevaba crampones"
- "Hacía un viento de 80km/h en la cresta, era peligroso"
- Planificación de equipo y vestimenta antes de salir

## 👤 Experiencia de usuario

1. En la ficha de la ruta, pestaña "Condiciones" con iconos intuitivos
2. Badges de color en listados: 🟢 Buenas / 🟡 Precaución / 🔴 Adversas
3. Detalle por tramo (inicio, medio, cima) con altitudes distintas
4. Pronóstico a 5 días para planificar

## ⚙️ Requisitos técnicos

- [ ] API `GET /api/conditions/trail?lat=&lng=&elevation=`
- [ ] Integración OpenWeather One Call API 3.0
- [ ] Integración AEMET (datos oficiales España)
- [ ] Caché de condiciones (TTL 1 hora, `unstable_cache`)
- [ ] Cálculo de condiciones por altitud (gradiente térmico -6.5°C/1000m)

## 📊 Criterios de éxito

- Datos actualizados cada hora
- Precisión de temperatura ±2°C
- Cobertura: 100% de rutas con coordenadas

## 🔗 Dependencias

- Coordenadas de la ruta (lat/lng del modelo Track)
- API keys de OpenWeather y/o AEMET
- Plan Peak
