## 🎯 Descripción funcional

El usuario fotografía una planta, flor o árbol y la app identifica la especie usando modelos de visión artificial. Muestra nombre común, nombre científico, hábitat, época de floración y si es tóxica o comestible.

## 🧠 Problema que resuelve

- "¿Qué planta es esa? ¿Es venenosa?"
- Enriquecimiento educativo de la experiencia en natura
- Feature diferenciadora (ningún competidor la ofrece integrada)

## 👤 Experiencia de usuario

1. Pulsar "Identificar planta" → abre la cámara
2. Enfocar la planta y tomar foto
3. Procesamiento (1-3 segundos)
4. Resultado: nombre, foto de referencia, descripción, ⚠️ toxicidad
5. Opción de guardar en "Mi herbario" personal

## ⚙️ Requisitos técnicos

- [ ] Modelo de clasificación: PlantNet API o modelo propio (TFLite)
- [ ] BD `Flora` (name, scientificName, imageUrl, habitat, toxic, edible)
- [ ] Procesamiento: preferiblemente on-device para funcionar offline
- [ ] Fallback: enviar a servidor si modelo local no tiene confianza >80%
- [ ] Historial de identificaciones por usuario

## 📊 Criterios de éxito

- Precisión de identificación >85% (top-3)
- Tiempo de respuesta <3 segundos (on-device)
- BD de flora: >500 especies ibéricas

## 🔗 Dependencias

- Modelo de IA pre-entrenado en flora ibérica
- Cámara del dispositivo
- Plan Peak
