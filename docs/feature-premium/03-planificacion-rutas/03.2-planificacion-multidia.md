## 🎯 Descripción funcional

Herramienta para planificar travesías de varios días: encadenar etapas, marcar refugios/alojamientos, calcular tiempos acumulados y gestionar puntos de avituallamiento. Ideal para GR completos, Camino de Santiago, Alta Ruta Pirenaica.

## 🧠 Problema que resuelve

- "Quiero hacer el GR11 en 10 días, ¿cómo divido las etapas?"
- Planificación logística: dónde dormir, dónde reponer agua/comida
- Cálculo realista de tiempos con fórmula de Naismith + descansos

## 👤 Experiencia de usuario

1. Seleccionar ruta larga (o combinar varias rutas)
2. Dividir en etapas diarias arrastrando marcadores
3. Para cada etapa: ver distancia, desnivel, duración estimada
4. Marcar refugios/alojamientos/fuentes en cada etapa
5. Exportar plan completo como PDF o lista de waypoints

## ⚙️ Requisitos técnicos

- [ ] Editor visual de etapas (split & merge de stages)
- [ ] Fórmula de tiempos: Naismith + factor de fatiga por día
- [ ] Modelo `MultiDayPlan` (userId, name, stages[], startDate)
- [ ] Base de datos de refugios/albergues con disponibilidad
- [ ] Exportar a PDF / iCal / GPX multi-etapa

## 📊 Criterios de éxito

- Creación de plan multidía en <5 minutos
- Tiempos estimados ±15% de realidad
- Exportación correcta a todos los formatos

## 🔗 Dependencias

- Tracks/stages existentes con geometría
- BD de refugios y alojamientos
- Plan Explorer o superior
