## 🎯 Descripción funcional

Exportar rutas directamente a dispositivos GPS (Garmin, Suunto) y relojes deportivos (Apple Watch, Garmin Watch) en formatos compatibles. Integración con Garmin Connect y Apple HealthKit para sincronización automática.

## 🧠 Problema que resuelve

- "Quiero usar la ruta en mi Garmin, pero no puedo exportar el GPX"
- Navegación con dispositivo especializado (mejor batería que el móvil)
- Integración con el ecosistema deportivo existente del usuario

## 👤 Experiencia de usuario

1. En la ficha de ruta, botón "Exportar"
2. Opciones: GPX, KML, FIT (Garmin), enviar a Garmin Connect, enviar a Apple Watch
3. Para Garmin Connect: OAuth2 → envío directo al reloj
4. Para Apple Watch: via la app companion de GuideFlow
5. Confirmación de envío exitoso

## ⚙️ Requisitos técnicos

- [ ] Generador GPX/KML/FIT desde modelo Track
- [ ] Integración Garmin Connect API (OAuth2 + Course Upload)
- [ ] Integración Apple HealthKit / Apple Watch (via React Native)
- [ ] API `GET /api/premium/export/:trackId?format=gpx|kml|fit`
- [ ] Rate limiting para exportaciones (evitar abuso)

## 📊 Criterios de éxito

- Exportación GPX en <2 segundos
- Compatibilidad verificada con Garmin Fenix, Suunto 9, Apple Watch Ultra
- Sincronización automática con Garmin Connect en <30 segundos

## 🔗 Dependencias

- Track.gpxData existente en BD
- API keys de Garmin Connect
- Plan Explorer o superior
