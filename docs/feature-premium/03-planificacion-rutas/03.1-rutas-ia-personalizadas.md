## 🎯 Descripción funcional

Motor de recomendación que genera rutas personalizadas según las preferencias del usuario: distancia deseada, desnivel, tipo de terreno, duración, dificultad y zona. Combina segmentos de rutas existentes y senderos GR/PR/SL para crear itinerarios a medida.

## 🧠 Problema que resuelve

- "Quiero una ruta de 15km con 800m de desnivel por sendero, ¿cuál me recomiendas?"
- Descubrir rutas que se ajustan exactamente a tu nivel y tiempo disponible
- Combinación inteligente de tramos para crear experiencias únicas

## 👤 Experiencia de usuario

1. "Planificar ruta con IA" → formulario de preferencias
2. Inputs: zona, distancia (±), desnivel (±), dificultad, circular/lineal, duración
3. La IA propone 3 opciones con mapa, perfil y justificación
4. El usuario elige una y la guarda como ruta personalizada
5. Puede ajustar waypoints manualmente tras la generación

## ⚙️ Requisitos técnicos

- [ ] Grafo de senderos: nodos (intersecciones) + aristas (tramos con metadata)
- [ ] Algoritmo de path-finding con restricciones (A* modificado o genetic)
- [ ] API `POST /api/premium/routes/generate` con parámetros de preferencia
- [ ] Uso de tracks y stages existentes como base de datos de tramos
- [ ] Scoring de rutas generadas: variedad paisajística, puntos de interés, circularidad

## 📊 Criterios de éxito

- Generación de ruta en <10 segundos
- Satisfacción del usuario >4/5 estrellas en la ruta generada
- Mínimo 3 opciones diferentes por consulta

## 🔗 Dependencias

- Base de datos de tracks/stages con geometría completa
- Grafo de senderos (generado desde GPX existentes)
- Plan Peak
