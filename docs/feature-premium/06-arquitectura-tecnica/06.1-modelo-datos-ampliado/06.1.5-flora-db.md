## 📊 Modelo Prisma propuesto

```prisma
model Flora {
  id             String  @id @default(uuid())
  name           String  // Nombre común
  scientificName String  // Nombre científico
  family         String? // Familia botánica
  imageUrl       String
  habitat        String? // Bosque, pradera, roca, etc.
  altitude       String? // Rango de altitud "500-2000m"
  flowering      String? // Meses de floración "mayo-julio"
  toxic          Boolean @default(false)
  edible         Boolean @default(false)
  protected      Boolean @default(false)
  description    String? @db.Text
  region         String? // Zona geográfica

  @@index([scientificName])
  @@index([name])
}
```

## 🎯 Propósito

Base de datos de flora ibérica para el identificador de plantas con IA. Objetivo: 500+ especies de plantas, árboles y flores comunes en senderos españoles.

## ⚙️ Fuentes de datos

- GBIF (Global Biodiversity Information Facility)
- Flora Iberica (CSIC)
- PlantNet (referencia cruzada)
- Importación semi-automática con validación manual
