## 📊 Modelo Prisma propuesto

```prisma
model OfflineMap {
  id        String   @id @default(uuid())
  userId    String
  name      String   // "Pirineos Centrales"
  bbox      Json     // {minLat, minLng, maxLat, maxLng}
  zoomMin   Int      @default(10)
  zoomMax   Int      @default(16)
  url       String   // URL del paquete MBTiles en S3
  sizeMB    Float    // Tamaño en MB
  expiresAt DateTime // 30 días desde descarga
  createdAt DateTime @default(now())

  user AppUser @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@index([userId])
  @@index([expiresAt])
}
```

## 🎯 Propósito

Registro de paquetes de mapas offline descargados por cada usuario. Permite gestionar caducidad, limitar descargas por plan y limpiar archivos expirados.

## ⚙️ Consideraciones

- Explorer: máx 3 zonas descargadas simultáneamente
- Peak: zonas ilimitadas
- Job cron para limpiar paquetes expirados de S3
- Tamaño medio por zona: 30-80MB (zoom 10-16)
