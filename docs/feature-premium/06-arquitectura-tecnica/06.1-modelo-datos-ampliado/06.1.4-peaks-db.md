## 📊 Modelo Prisma propuesto

```prisma
model Peak {
  id          String  @id @default(uuid())
  name        String
  altName     String? // Nombre alternativo (catalán, euskera, etc.)
  lat         Float
  lng         Float
  elevation   Int     // Metros
  prominence  Int?    // Prominencia en metros
  region      String  // "Pirineos", "Sierra Nevada", etc.
  province    String?
  country     String  @default("España")
  description String? @db.Text
  imageUrl    String?
  osmId       String? // ID de OpenStreetMap para referencia

  @@unique([lat, lng])
  @@index([region])
  @@index([elevation])
}
```

## 🎯 Propósito

Base de datos de cimas para el identificador de picos con Realidad Aumentada. Contiene todas las cimas con nombre y altitud de España (+2000 inicialmente).

## ⚙️ Fuentes de datos

- OpenStreetMap (natural=peak)
- IGN (Instituto Geográfico Nacional)
- Base de datos Munros/Peaks de FEDME
- Importación masiva con script de Overpass API
