## 📊 Modelo Prisma propuesto

```prisma
model TrafficData {
  id        String   @id @default(uuid())
  lat       Float
  lng       Float
  h3Index   String   // Celda H3 (resolución 8)
  timestamp DateTime @default(now())
  anonId    String?  // Hash anónimo del userId

  @@index([h3Index, timestamp])
  @@index([lat, lng])
}
```

## 🎯 Propósito

Almacenar posiciones GPS anónimas para generar mapas de calor de afluencia en rutas. Los datos se agregan por celda H3 para pronósticos de tráfico.

## ⚙️ Consideraciones

- Anonimización: hash irreversible del userId
- Retención: datos raw 90 días, agregados indefinidamente
- Volumen estimado: ~1M registros/mes con 10K usuarios activos
- Índice H3 resolución 8 ≈ 460m² por celda (adecuado para senderos)
