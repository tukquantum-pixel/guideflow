## 🎯 Descripción

Generador de mapas de calor de tráfico a partir de datos GPS anónimos. Procesa los `TrafficData` en batch nocturno para crear pronósticos de afluencia por hora/día.

## ⚙️ Pipeline

1. **Ingest**: posiciones GPS anónimas → tabla `TrafficData`
2. **Aggregate**: agrupar por celda H3 (res 8) + hora + día de semana
3. **Predict**: modelo estadístico (media móvil 4 semanas) por celda/hora
4. **Cache**: resultados en tabla `TrafficPrediction` con TTL 24h
5. **Serve**: API `GET /api/premium/traffic` lee de caché

## 📊 Modelo de datos de salida

```typescript
interface TrafficPrediction {
  h3Index: string
  dayOfWeek: number  // 0-6
  hour: number       // 0-23
  intensity: number  // 0-100
  sampleSize: number
}
```
