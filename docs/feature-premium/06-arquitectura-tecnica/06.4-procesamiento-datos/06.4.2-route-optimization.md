## 🎯 Descripción

Algoritmo de generación y optimización de rutas personalizadas con IA. Construye un grafo de senderos a partir de tracks existentes y genera rutas que cumplen restricciones del usuario.

## ⚙️ Pipeline

1. **Graph Build**: convertir tracks/stages en grafo dirigido de nodos + aristas
2. **Constraint Parsing**: interpretar preferencias (distancia, desnivel, terreno, circular)
3. **Path Search**: A* modificado con función de coste multi-criterio
4. **Scoring**: evaluar candidatos por variedad paisajística + POIs + circularidad
5. **Return**: top 3 rutas con mapa, perfil y justificación

## 📊 Modelo del grafo

```typescript
interface TrailNode {
  id: string
  lat: number
  lng: number
  elevation: number
  edges: TrailEdge[]
}

interface TrailEdge {
  targetId: string
  distance: number
  elevationGain: number
  terrain: string
  difficulty: number
}
```
