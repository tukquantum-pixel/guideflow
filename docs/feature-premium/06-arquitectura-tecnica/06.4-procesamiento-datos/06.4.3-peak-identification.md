## 🎯 Descripción

Algoritmo de identificación de cimas visibles desde una posición dada. Calcula línea de vista considerando terreno, curvatura terrestre y refracción atmosférica.

## ⚙️ Algoritmo

1. **Input**: posición usuario (lat, lng, alt), heading, pitch, FOV
2. **Query**: peaks en radio de 50km desde posición
3. **Line-of-sight**: para cada peak, verificar si hay terreno bloqueante
4. **Projection**: calcular ángulo azimuth y elevación visual
5. **Filter**: peaks dentro del FOV de la cámara
6. **Output**: lista de peaks visibles con coordenadas de pantalla

## 📊 DEM (Digital Elevation Model)

- Fuente: SRTM 30m (NASA) o ASTER GDEM
- Pre-procesado y almacenado como tiles en S3
- Resolución suficiente para line-of-sight a 50km
