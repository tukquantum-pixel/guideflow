## 🎯 Descripción

GPS tracking en background con optimización de batería. Core de las funcionalidades premium: alertas de desvío y live tracking dependen de esto.

## 📊 Librería: `expo-location`

```typescript
// Configuración de tracking
const config = {
  accuracy: Location.Accuracy.High,
  timeInterval: 5000,      // cada 5 segundos
  distanceInterval: 10,     // o cada 10 metros
  foregroundService: {
    notificationTitle: "GuideFlow navegando",
    notificationBody: "Tracking activo"
  }
}
```

## ⚙️ Modos de operación

| Modo | Intervalo | Batería | Uso |
|------|-----------|---------|-----|
| Navegación activa | 5s | Alto | Alertas desvío |
| Live tracking | 30s | Medio | Compartir ubicación |
| Background pasivo | 60s | Bajo | Recolección tráfico (anónimo) |

## 📊 Optimizaciones

- Kalman filter para suavizar posiciones GPS
- Batch de puntos para envío al servidor (cada 5 min o al recuperar conexión)
- Geofencing para alertas de proximidad a checkpoints
