## 🎯 Descripción

Almacenamiento local en el dispositivo para mapas, tracks GPX, waypoints y datos de sesión. Funcionalidad central para que la app funcione sin conexión.

## 📊 Stack de almacenamiento

| Dato | Tecnología | Tamaño estimado |
|------|-----------|-----------------|
| Map tiles | SQLite (MBTiles) | 30-80MB/zona |
| Tracks GPX | AsyncStorage/MMKV | <1MB/ruta |
| Waypoints offline | WatermelonDB | <1MB |
| Fotos pendientes | FileSystem | Variable |

## ⚙️ Sincronización

- Patrón: local-first, sync al tener conexión
- Resolución de conflictos: last-write-wins con timestamp
- Cola de operaciones pendientes (create/update/delete)
- Indicador de sync status en la UI
