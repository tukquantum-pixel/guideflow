## 🎯 Descripción

Visualización AR para identificar picos montañosos. Superpone información de cimas sobre la vista de la cámara usando GPS + brújula + pitch del dispositivo.

## 📊 Stack técnico

| Componente | Tecnología |
|-----------|-----------|
| AR Framework | ViroReact o expo-three |
| Sensores | expo-sensors (magnetómetro, acelerómetro) |
| Cálculo de visibilidad | Algoritmo line-of-sight con DEM |
| Base de cimas | Modelo Peak (06.1.4) |

## ⚙️ Algoritmo simplificado

1. Obtener posición GPS + altitud del usuario
2. Calcular heading (brújula) y pitch (inclinómetro)
3. Consultar peaks en cono de visión (heading ±30°, distancia <50km)
4. Para cada peak: calcular ángulo vertical esperado
5. Renderizar etiqueta AR en la posición correspondiente

## 📊 Requisitos de dispositivo

- iPhone 8+ / Android con ARCore
- Giroscopio + magnetómetro
- Fase 3 de desarrollo (complejidad alta)
