## 🎯 Integración

Sincronización de rutas con Apple Watch a través de HealthKit y WatchConnectivity. Permite enviar rutas al reloj y recibir datos de sesión (HR, calorías, distancia).

## 📊 Frameworks a usar

| Framework | Uso |
|-----------|-----|
| HealthKit | Leer/escribir datos de salud y fitness |
| WatchConnectivity | Comunicación iPhone ↔ Apple Watch |
| MapKit on Watch | Mostrar ruta en la pantalla del reloj |

## ⚙️ Implementación

- React Native: `react-native-health` para HealthKit
- App companion WatchOS (nativa Swift, invocada via bridge)
- Envío de waypoints simplificados al reloj (pantalla pequeña)
- Lectura de HR/calorías post-ruta para resumen de actividad

## 🔑 Requisitos

- Cuenta Apple Developer ($99/año)
- Entitlements de HealthKit en el bundle
- Plan Explorer o superior
