## 🎯 Integración

OpenWeather One Call API 3.0 para condiciones meteorológicas en tiempo real y pronóstico a 5 días.

## 📊 Endpoints a usar

| Endpoint | Uso | Coste |
|----------|-----|-------|
| `onecall?lat=&lon=` | Condiciones actuales + 7 días | 1 call = 1 crédito |
| `onecall/timemachine?lat=&lon=&dt=` | Histórico | 1 call = 1 crédito |

## ⚙️ Implementación

- Caché TTL 1h con `unstable_cache` de Next.js
- Gradiente térmico: -6.5°C por 1000m de altitud
- Plan "One Call by Call": pago por uso, ~0.001$/call
- Estimación: 10 calls/ruta/día × 200 rutas = 2000 calls/día ≈ 60$/mes

## 🔑 Config

```env
OPENWEATHER_API_KEY=xxx
OPENWEATHER_BASE_URL=https://api.openweathermap.org/data/3.0
```
