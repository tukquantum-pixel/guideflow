## 🎯 Integración

Datos de calidad del aire para zonas de ruta, relevante para usuarios con alergias o sensibilidad respiratoria. Complemento a las condiciones meteorológicas.

## 📊 Endpoints

| API | Datos | Coste |
|-----|-------|-------|
| OpenWeather Air Pollution | AQI, PM2.5, PM10, O3, NO2 | Incluido en One Call |
| AEMET | Índice de polen por región | Gratuito |

## ⚙️ Implementación

- Caché TTL 3h (la calidad del aire cambia lentamente)
- Iconos: 🟢 Buena / 🟡 Moderada / 🔴 Mala / ⚫ Peligrosa
- Badge en la ficha de ruta junto a condiciones meteorológicas
- Alerta si AQI >100 ("No recomendable para ejercicio intenso")
