## 🎯 Integración

Sincronización de rutas con dispositivos Garmin a través de Garmin Connect API. Permite enviar rutas directamente al reloj/GPS del usuario.

## 📊 Endpoints a usar

| Endpoint | Uso |
|----------|-----|
| OAuth2 Connect | Autorización del usuario |
| `POST /course-api/course` | Subir ruta como curso |
| `GET /wellness-api/wellness` | Leer datos de actividad |

## ⚙️ Implementación

- OAuth2 flow: GuideFlow → Garmin → callback con token
- Convertir GPX a formato FIT (Garmin nativo)
- Librería: `@garmin-fit/sdk` para generación de archivos FIT
- Almacenar refresh token del usuario para sincronización futura

## 🔑 Requisitos

- Registro como partner en Garmin Developer Portal
- API key aprobada (proceso de 2-4 semanas)
- Plan Explorer o superior
