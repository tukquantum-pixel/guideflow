## 🎯 Marco legal

El tratamiento de datos GPS está sujeto al **RGPD** (Reglamento General de Protección de Datos) y la **LOPDGDD** (Ley Orgánica de Protección de Datos española).

## ⚖️ Requisitos legales

| Requisito | Implementación |
|-----------|---------------|
| **Base legal** | Consentimiento explícito (Art. 6.1.a RGPD) |
| **Finalidad** | Navegación, alertas, live tracking (especificar cada uso) |
| **Minimización** | Solo lat/lng/timestamp, no velocidad/heading innecesarios |
| **Retención** | Live tracking: 24h. Tráfico anónimo: 90 días. Waypoints: indefinido |
| **Portabilidad** | Exportar todos los datos GPS en formato GPX estándar |
| **Supresión** | Borrar todos los datos GPS a petición del usuario |

## ⚙️ Anonimización para tráfico

- Hash SHA-256 del userId → anonId irreversible
- Aggregación por celda H3 (res 8, ~460m²) → imposible re-identificar
- No almacenar trayectorias individuales, solo puntos discretos
