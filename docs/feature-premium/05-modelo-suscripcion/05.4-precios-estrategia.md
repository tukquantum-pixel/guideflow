## 🎯 Estrategia de precios

### Principios

1. **Accesibilidad**: precio inferior a competencia (AllTrails 35,99$, Komoot ~60€)
2. **Anualidad**: solo pago anual para simplificar y fidelizar
3. **Diferenciación clara**: Explorer = seguridad, Peak = datos avanzados

### Análisis de precios del mercado

| App | Plan básico | Plan avanzado | Modelo |
|-----|-------------|---------------|--------|
| AllTrails+ | 35,99$/año | 59,99$/año (Plus+) | Anual |
| Komoot | ~60€ (por regiones) | - | Compra única por región |
| Wikiloc | 9,99€/año | 19,99€/año | Anual |
| Strava | 79,99$/año | - | Anual |
| **GuideFlow Explorer** | **9,99€/año** | - | Anual |
| **GuideFlow Peak** | - | **19,99€/año** | Anual |

### Justificación de precios

- **Explorer 9,99€/año = 0,83€/mes**: más barato que un café. Barrera de entrada mínima.
- **Peak 19,99€/año = 1,67€/mes**: al nivel de Wikiloc Premium, pero con más funcionalidades.
- **Ventaja vs Komoot**: modelo "todo incluido" vs pago por región.
