## 🎯 Descripción funcional

Gestión de suscripciones B2C a través de Stripe Checkout. El flujo ya está parcialmente implementado: creación de checkout session, webhook para activación de plan y modelo de datos con `stripeCustomerId` y `stripeSubId`.
