## 🎯 KPI: Feature Adoption Rate

**Fórmula**: `(usuarios que usan feature X / usuarios con acceso a feature X) × 100`

## 📊 Métricas por funcionalidad

| Feature | Target adoption | Frecuencia medición |
|---------|----------------|---------------------|
| Mapas offline | >70% de Explorer | Semanal |
| Alertas desvío | >50% de Explorer | Semanal |
| Live tracking | >30% de Explorer | Semanal |
| Exportar GPS | >20% de Explorer | Mensual |
| Condiciones sendero | >60% de Peak | Semanal |
| Identificador cimas | >40% de Peak | Mensual |

## ⚙️ Implementación de tracking

- Eventos: `feature_used:{feature_name}`, `feature_first_use:{feature_name}`
- Cohorte: segmentar por plan, antigüedad, zona geográfica
- Dashboard: distribución de uso por feature → identificar features infrautilizadas
