## 🎯 KPI: Valor de vida del cliente

**Fórmula**: `ARPU × (1 / churn rate)`

## 📊 Proyecciones

| Plan | ARPU/año | Churn estimado | LTV |
|------|----------|----------------|-----|
| Explorer | 9,99€ | 15% | 66,60€ |
| Peak | 19,99€ | 12% | 166,58€ |
| Blended | 13,33€ | 14% | 95,21€ |

## ⚙️ Estrategias para aumentar LTV

1. **Upsell Explorer → Peak**: automatizar oferta tras 3 meses
2. **Retención**: reducir churn con contenido exclusivo
3. **Cross-sell**: ofrecer experiencias con guías (comisión por reserva)
4. **Engagement**: retos y gamificación para aumentar frecuencia de uso

## 📊 CAC vs LTV

- **CAC objetivo**: <15€ (orgánico + SEO + referral)
- **Ratio LTV/CAC**: >3x (95€/15€ = 6.3x) → saludable
