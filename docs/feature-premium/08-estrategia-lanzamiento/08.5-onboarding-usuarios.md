## 🎯 Flujo de onboarding

### Primer uso (app)

1. **Splash** → logo + "Tu compañero de montaña"
2. **Permisos** → GPS (explicando por qué), notificaciones
3. **Perfil rápido** → nombre, zona, nivel (principiante/intermedio/experto)
4. **Descubrir** → mostrar 3 rutas cerca del usuario
5. **CTA suave** → "Desbloquea mapas offline con Explorer - 7 días gratis"

### Primer ruta (momento clave)

1. Seleccionar ruta → ver info + descarga offline (🔒 si gratis)
2. Iniciar navegación → experiencia completa (trial o premium)
3. Post-ruta → resumen + fotos + "¿Te gustó? Valora la ruta"
4. Si trial → "Te quedan X días de Explorer gratuito"

## 📊 Métricas de onboarding

| Paso | Target conversión |
|------|-------------------|
| Instalación → registro | >60% |
| Registro → primera ruta | >40% |
| Primera ruta → segunda ruta | >30% |
| Trial → pago | >15% |
