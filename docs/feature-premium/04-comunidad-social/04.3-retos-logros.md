## 🎯 Descripción funcional

Sistema de gamificación con retos (completar X rutas, acumular Y km de desnivel, visitar Z cimas) y logros desbloqueables. Incluye clasificaciones por zona, temporada y nivel de dificultad. Los logros se pueden compartir en redes sociales.

## 🧠 Problema que resuelve

- Motivación continuada para usar la app regularmente
- Descubrimiento de nuevas rutas a través de retos
- Fidelización y reducción de churn

## 👤 Experiencia de usuario

1. Sección "Mis logros" en el perfil del usuario
2. Retos activos: "Completa 5 GR este mes" → barra de progreso
3. Al completar un reto → badge animado + notificación
4. Clasificaciones: "Top senderistas de Cataluña este mes"
5. Compartir logro en Instagram/Twitter con imagen generada

## ⚙️ Requisitos técnicos

- [ ] Modelo `Challenge` (name, description, criteria, reward, startDate, endDate)
- [ ] Modelo `UserAchievement` (userId, challengeId, progress, completedAt)
- [ ] Motor de evaluación: ¿cumplió los criterios? (batch job nocturno)
- [ ] Generador de imágenes para compartir (canvas server-side)
- [ ] Sistema de clasificaciones con cache (top 100 por zona)

## 📊 Criterios de éxito

- >30% de usuarios activos participan en retos
- Aumento de frecuencia de uso 1.5x con gamificación
- >10% comparten logros en redes sociales

## 🔗 Dependencias

- Tracking de rutas completadas por usuario
- Sistema de notificaciones
- Fase 3 de desarrollo
