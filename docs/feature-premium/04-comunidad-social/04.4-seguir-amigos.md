## 🎯 Descripción funcional

Red social ligera para seguir a amigos, ver sus rutas completadas, fotos y logros. Feed de actividad tipo Strava pero centrado en senderismo. Posibilidad de planificar rutas juntos y compartir condiciones.

## 🧠 Problema que resuelve

- "¿Qué rutas ha hecho mi amigo últimamente?"
- Descubrimiento de rutas a través de contactos de confianza
- Motivación social para salir más al monte

## 👤 Experiencia de usuario

1. Buscar amigos por email o username
2. Enviar solicitud de seguimiento
3. Feed: "María completó la Ruta del Cares hace 2h" + foto
4. Ver estadísticas comparadas (km totales, desnivel acumulado)
5. "Planificar ruta juntos" → ruta compartida editable

## ⚙️ Requisitos técnicos

- [ ] Modelo `Follow` (followerId, followingId, createdAt)
- [ ] Modelo `ActivityFeed` (userId, type, data, createdAt)
- [ ] API `/api/social/follow`, `/api/social/feed`
- [ ] Privacidad: perfil público/privado, control de visibilidad
- [ ] Notificaciones de nueva actividad de personas seguidas

## 📊 Criterios de éxito

- Media de 5 conexiones por usuario activo
- >3 interacciones por semana por usuario con red social
- 0 incidencias de privacidad

## 🔗 Dependencias

- Sistema de autenticación de AppUser
- Sistema de notificaciones
- Fase 3 de desarrollo
