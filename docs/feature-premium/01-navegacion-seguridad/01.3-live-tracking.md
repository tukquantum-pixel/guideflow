## 🎯 Descripción funcional

Comparte tu ubicación en tiempo real con contactos de confianza durante una ruta. Genera un enlace único que permite a familiares/amigos ver tu posición en un mapa web sin necesidad de instalar la app. La sesión expira automáticamente al finalizar la ruta.

## 🧠 Problema que resuelve

- "Mi familia no sabía dónde estaba y se preocupaban"
- Seguridad para senderistas que van solos
- Facilitador de rescate en caso de emergencia

## 👤 Experiencia de usuario

1. Antes de salir, el usuario pulsa "Compartir ubicación"
2. Se genera un enlace (ej: `guideflow.com/live/abc123`)
3. Envía el enlace por WhatsApp/SMS a sus contactos
4. Los contactos ven un mapa con la posición actualizada cada 30s
5. Al terminar la ruta, la sesión se cierra automáticamente

## ⚙️ Requisitos técnicos

- [ ] API `POST /api/premium/live-tracking/start` → genera token único
- [ ] API `GET /api/premium/live-tracking/share/:token` → página pública con mapa
- [ ] WebSocket o polling cada 30s para actualizar posición
- [ ] Modelo `LiveSession` (userId, shareToken, routeId, startedAt, expiresAt)
- [ ] GPS background con envío periódico al servidor

## 📊 Criterios de éxito

- Latencia de actualización <45 segundos
- Enlace funcional sin login del receptor
- Expiración automática tras 24h o fin de ruta

## 🔗 Dependencias

- GPS en background (app móvil)
- Conexión a internet (al menos intermitente)
- Plan Explorer o superior

## 🧩 Notas adicionales

- Fallback: si pierde conexión, bufferea posiciones y envía cuando recupera señal
- Considerar botón de emergencia SOS en la interfaz de tracking
