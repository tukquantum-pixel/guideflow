## 🎯 Descripción funcional

Sistema de alertas automáticas que notifica al usuario cuando se aleja del sendero planificado. Utiliza la posición GPS del dispositivo y la compara continuamente con el track GPX de la ruta seleccionada. Si la distancia supera un umbral configurable (por defecto 50m), se emite una alerta sonora y vibración.

## 🧠 Problema que resuelve

- "Me salí del sendero sin darme cuenta y perdí 2 horas volviendo al camino"
- Crítico para rutas con bifurcaciones poco señalizadas
- Especialmente valioso en condiciones de baja visibilidad (niebla, noche)

## 👤 Experiencia de usuario

1. El usuario selecciona una ruta y pulsa "Iniciar navegación"
2. El GPS del móvil compara su posición con el track cada 5 segundos
3. Si se desvía >50m → alerta suave (vibración + aviso visual)
4. Si se desvía >200m → alerta fuerte (sonido + notificación push)
5. El usuario puede configurar umbrales en ajustes

## ⚙️ Requisitos técnicos

- [ ] Algoritmo de distancia punto-a-polilínea (Haversine + proyección)
- [ ] GPS en background (React Native, permisos de ubicación)
- [ ] Notificaciones push locales (sin servidor)
- [ ] Configuración de umbrales por usuario
- [ ] Bajo consumo de batería (optimizar intervalos de lectura GPS)

## 📊 Criterios de éxito

- Detección de desvío en <10 segundos
- Precisión GPS ±10m en condiciones normales
- Consumo de batería <5%/hora en navegación activa

## 🔗 Dependencias

- Track GPX cargado en memoria (modelo `Track.gpxData`)
- Permisos de ubicación en background (app móvil)
- Plan Explorer o superior

## 🧩 Notas adicionales

- Consideración offline: toda la lógica debe funcionar sin conexión
- La alerta debe funcionar incluso con la pantalla apagada
